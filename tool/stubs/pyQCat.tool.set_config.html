save_config_to_store(config_dir=None)¶

Save batch config file to MongoDB ConfigStore. Support the file postfix with dat, json, bin. Just save one object about file name to ConfigStore. First, query by file name, if exist update, else create a new object.

参数

config_dir – dir name of config files

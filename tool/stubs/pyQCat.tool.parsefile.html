get_iq_discriminator(file, qubit, union=None)¶

获取IQ分类器

首先会从数据库中读取，如果没有读到，则从指定的配置文件中加载

参数

  • file (str) – 配置文件路径

  • qubit (int) – 比特编号

  • union (Optional[List]) – 获取联合读取判据时，必须传入 union 参数

返回类型

IQdiscriminator

get_inst(file, bit=8)¶

获取初始设备对象

默认从本地配置文件中加载

参数

  • file (str) – 配置文件路径

  • bit (int) – 设备位数，支持8位和30位

引发

ValueError – bit 属性既不是 8 也不是 30

返回类型

Instrument

返回

设备对象

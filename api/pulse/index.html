Quantum Circuits (pyQCat.pulse)¶

pyQCat.pulse 模块定义了实验常用的波形，你可以通过它构造你想要的波形。

Base Pulse¶

PulseComponent

波形的抽象基类

PulseCorrection

波形修正类

Extend Pulse¶

Drag

Drag波形， 作用于 XY 线

GaussianSquare

高斯方波，作用于 Z 线

FlatTopGaussian

FlatTopGaussian波形， 作用于 Z 线

AcquireSine

AcquireSine， 作用于读取通道

PulseComponent

波形的抽象基类

Constant

Constant 波形，作用于 Z 线

SquareEnvelop

SquareEnvelop 波形，又称方波，作用于 XY 线

half_pi_pulse

获取 half pi pulse 将比特激发到赤道平面

pi_pulse

获取 pi pulse 将比特激发到1态

zero_pulse

获取幅值为一直为0的波形

Pulse Tool¶

Schedule

绘制波形时序图功能类

基类：pyQCat.experiment.top_experiment.TopExperiment

腔功率（读取功率）优化实验

腔功率优化实验用来获取读取功率，即 Qubit 对象的 probe_power 属性

  • 输入扫描读取功率范围，假设为[ -34, -10, 2 ]；

  • 遍历读取功率，执行SingleShot实验，每次实验重复5000次，结果取5000次实验均值；

  • 获取每个读取功率下SingleShot实验结果

  • 交给结果分析类进行分析

run(power_list: List, save_bin: bool = False)¶

读取功率优化实验运行接口

参数

  • power_list – 扫描的读取功率列表

  • save_bin – 传入引用的SingleShot实验，决定分类器是否保存

实际案例

>>> inst = get_parameters("instrument")
>>> qubit = get_parameters("qubit", bit)
>>> dc = corrected_dc(6)
>>> compensate = get_parameters("compensate", bit)
# 获取设备、比特、通道DC电压、脉冲补偿参数构造实验
>>> cp = CavityPowerOptimize(inst, qubit, correct_dc=dc, compensate=compensate)
# 执行腔功率优化实验
>>> cp.run(power_list)

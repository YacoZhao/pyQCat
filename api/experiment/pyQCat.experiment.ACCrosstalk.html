基类：object

AC 串扰实验

假设有两个比特分别为bias qubit和target qubit。 串扰，顾名思义就是bias qubit的flux pulse不仅能调节其自己的频率， 还能影响芯片上其他比特，如target qubit的频率。因此在实验中必须进行串扰实验， 以此计算出相邻比特之间的串扰系数（影响程度）。首先通过AC谱实验测量target qubit 自身flux pulse对比特频率的调制，得到AC谱参数，然后再测量bias qubit的flux pulse 对target qubit的频率调制，拟合得到的MB。有了target qubit自身的调制谱，再结合 bias qubit的flux pulse对target qubit的调制，可以直接计算出Vb相当于多少Vt， 则串扰系数可知。这时要求比特处于对flux pulse幅度比较敏感的位置，因此在测bias qubit 的flux pulse对target qubit的调制时，可以同时在target qubit上施加等长的一定幅度的flux pulse。

run()¶

运行AC串扰实验并获取串扰系数

实验步骤：

  • 根据 freq_offset 属性计算将target_qubit的比特频率下调freq_offset所需要的flux pulse的幅值v_target

  • 用初始的init_v_bias加到偏置比特上，在当前的v_target下进行Ramsye实验，获取比特频率f10

  • 再根据target_qubit的AC谱参数计算不考虑串扰所需要的flux pulse幅值v_real

  • 根据v_target和v_real可以预估出一个串扰系数k0

  • 然后明确target_qubit比特频率的调制返回，根据freq_threshold确定

  • 根据串扰公式计算出目标比特进行评率调制所需的z_amp范围

  • 根据前面计算出来的k值，推测bias_qubit上施加的z_amp范围，记为v_bias_list

  • 遍历v_bias_list，施加到bias_qubit上，保持v_target施加的target_qubit上，

  • 记录每次实验用AC调制谱参数生成的v_real_list

  • 有了v_bias_list和v_real_list，就可以计算出AC串扰系数了

Examples:

from pyQCat.experiment.crosstalk import ACCrosstalk
from pyQCat.parameters import get_parameters, corrected_dc
from pyQCat.tool.utlis import qarange, connect_server, disconnect_server

def test_ac_crosstalk(target_bit: int, bias_bit: int,
                      init_fringe: float, delay_list: list):
    inst = get_parameters("instrument")
    target_qubit = get_parameters("qubit", target_bit)
    bias_qubit = get_parameters("qubit", bias_bit)
    ac_spectrum_params = get_parameters("fq_zamp", target_bit)
    compensate = get_parameters("compensate", target_bit)
    dc = corrected_dc(6, sweet_qubit=target_bit)
    dcm = get_parameters("dcm", target_bit)
    ac_exp = ACCrosstalk(inst, target_qubit, bias_qubit, init_fringe,
                         delay_list, ac_spectrum_params, correct_dc=dc,
                         compensate=compensate,discriminator=dcm)
    ac_exp.run()

if __name__ == '__main__':
    connect_server(ip="127.0.0.1")
    delay_list = qarange(200, 500, 5)
    init_fringe = -30
    test_ac_crosstalk(0, 1, init_fringe, delay_list)
    disconnect_server()
get_vbias_list(v_target: float, fringe: float, fd: float, IF: float)¶

获取偏置比特上的电压列表

参数

  • v_target (float) – 施加到目标比特上的电压值

  • fringe (float) – fringe频率，用来放大驱动频率和比特实际频率之间差异

  • fd (float) – 比特的驱动频率

  • IF (float) – 脉冲中频

run_ramsey(v_bias: float, v_target: float, fringe: float, fd: float, IF: float) → float¶

执行 Ramsey 实验获取施加到目标比特上的实际电压

参数

  • v_bias (float) – The voltage which was applied to bias qubit.

  • v_target (float) – The voltage which was applied to target qubit.

  • fringe (float) – Ramsey oscillation frequency which is used to amplify the difference between the drive frequency and the qubit real frequency.

  • fd (float) – The drive frequency of qubit.

  • IF (float) – The intermediate frequency of pulse.

返回

The real voltage on target qubit.

返回类型

v_real (float)

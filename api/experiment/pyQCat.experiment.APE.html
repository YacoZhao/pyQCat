基类：pyQCat.experiment.top_experiment.TopExperiment

APE 实验

利用 APE 实验可以对 XYWave 的 detune_pi 和 detune_pi2 参数进行标定。 DRAG波形在驱动频率上会引入一个失谐量 detune 来校准相位错误，提高门操作的保真度。 而APE实验正是用来校准所需施加的失谐量大小。APE实验通过增加插入的 X±π 对的数目， 可以提高实验结果对 detune 的敏感度。

run(name: str, sweep_list: List, N: int = 5, theta_type: str = 'Xpi', phi_num: int = 1, schedule_index: int = 0)¶

APE 实验启动函数

参数

  • name – 实验目的，可选detune和phase，其他选项则会报错

  • sweep_list – 实验扫描参数，可以的detune值扫描列表，也可以是phase值扫描列表

  • N – 波形重复次数

  • theta_type – 优化选项，可以为Xpi，也可以为XPi/2，其他选项将会报错

  • phi_num – todo

  • schedule_index – 绘制波形时序图的波形序号

引发

  • NameError – 如果name属性既不是detune也不是phase

  • ValueError – 如果theta_type属性既不是Xpi也不是XPi/2

Examples:

from pyQCat.experiment.ape import APE
from pyQCat.parameters import get_parameters, corrected_dc
from pyQCat.tool import qarange, remote_connection


@remote_connection("127.0.0.1")
def ape_run(bit: int, N, detune_list, theta):
    inst = get_parameters("instrument")
    qubit = get_parameters("qubit", bit)
    compensate = get_parameters("compensate", bit)
    dc = corrected_dc(6)
    dcm = get_parameters("dcm", bit)

    ape_exp = APE(inst, qubit, correct_dc=dc, compensate=compensate, discriminator=dcm, repeat=1000)
    ape_exp.run("detune", detune_list, N, theta)

    return ape_exp


if __name__ == "__main__":
    ape_run(2, 5 * 2, qarange(-10e-3, -3e-3, 0.1e-3), "Xpi/2")

excute_loop(func=None, args=None, *, analyze_one=None)¶

循环获取实验测量结果

参数

  • func – 一个回调函数，如果定义了，就用于单个循环获取数据， 并用作特殊情况下的替代接口。注意，这个函数必须返回结果和状态。

  • args – 回调函数 func 的参数

  • analyze_one – 一次循环的结果处理函数

func函数需满足下面的格式:

def func(*args):
    # do something
    return res, status

single_qubit_usualness()¶

单比特实验测量结果分析方法，如果 IQ分类器 为空，则返回振幅和相位测量参数。 否则，使用 IQ分类器 对采集的实验数据进行分类，返回状态0和状态1的概率。最终 将实验结果转化为 y0 和 y1 的形式

  • loop – 当前循环计数

返回

  • status 1: 以 amp-phase 的数据形式运行实验

  • status 2: 实验完成

  • status 3: 运行实验出错

  • status 4: 以 I-Q 的数据形式运行实验

返回类型

int

static slice_result(measure_result: List)¶

划分实验结果

它将从数据库 MeasureResultDoc 对象中获取的实验数据划分为 x、 y 的形式，如果取得的数据为 amp 和 phase，则 x 表示 amp， y表示 phase；否则 x 表示 I 值， y 表示 Q 值。

参数

measure_result – 由测量结果数据映射对象 MeasureResultDoc 组成的列表

返回

x 和 y 数据

返回类型

Tuple(List)

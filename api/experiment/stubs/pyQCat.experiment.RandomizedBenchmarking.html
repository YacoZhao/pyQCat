get_ancilla_gate(clifford_matrix_set, gate_matrix)¶

寻找逆元操作

根据群的定义，在 clifford 群中一定存在一个逆元使得这一系列操作等效为一个单位门，再把它施加到比特上。如果操作 都是完美的，比特此时应该仍然处于 0 态

参数

  • clifford_matrix_set – clifford 集群的矩阵形式

  • gate_matrix – 当前的连接门序列

返回

当亲矩阵逆元的字符串形式

返回类型

str

run(target_qubit=None, save_bin=True, is_check=True, path=None, acq_time=1500)¶

UnionSingleShot实验执行接口

参数

  • target_qubit (Optional[Qubit]) – 目标比特，如果提供了目标比特，则先给目标比特施加 zero_pulse ， 再施加 pi pulse ，其他比特仅施加 zero_pulse ； 如果为提供目标比特， 则对所有的比特，均先施加 zerp_pulse ，再施加 pi pulse

  • save_bin (bool) – 是否保存IQ分类器

  • is_check (bool) – 是否对分类结果进行检查

  • path (Optional[str]) – 保存路径

  • acq_time (int) – 读取波形时长

set_union_readout_params(acq_time)¶

设置联合读取参数

获取每个比特联合读取参数 amp 、 probe_IF 、 index ， 读取通道 任意挑选一个比特即可，因为他们的读取通道一致，将读取参数施加到联合读取通道，然后在 结合 acq_time 生成读取波形

参数

acq_time – 采样时长

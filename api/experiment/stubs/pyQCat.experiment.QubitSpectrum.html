pyQCat.experiment.QubitSpectrum¶

class QubitSpectrum(inst, *qubits, correct_dc=None, compensate=None, repeat=1000, bind_dc=True, bind_drive=True, bind_probe=True, discriminator=None, label=None, multi_readout_channels=None, schedule_type='sequence')¶

基类：pyQCat.experiment.top_experiment.TopExperiment

比特能谱实验

比特能谱实验用来获取比特频率.

  • 扫描XY线上的脉冲频率，在XY线上施加方波

  • 执行实验获取驱动频率 fq 和 amp-phase 的对应关系，然后进行寻峰操作

run(fq_list, power=None, z_amp=None, use_square=False)¶

比特能谱实验执行函数

参数

  • fq_list (Iterable) – 扫描参数(驱动频率 drive_freq)

  • power (Optional[float]) – 驱动功率

  • z_amp (Optional[float]) – Z线幅值

  • use_square (bool) – 是否使用方波，如果为 True ，则 XY 线上施加等频率的方波，如果为 False , 则 线上变频方波

Examples:

from pyQCat.experiment.qubit_spectrum import QubitSpectrum
from pyQCat.parameters import get_parameters, corrected_dc
from pyQCat.tool import qarange, remote_connection

@remote_connection("127.0.0.1")
def qs_test(bit, fq_list, power=-40, z_amp=None):
    inst = get_parameters("instrument")
    qubit = get_parameters("qubit", bit)
    qs_exp = QubitSpectrum(inst, qubit)
    qs_exp.run(fq_list, power, z_amp)

if __name__ == '__main__':
    zamp_list = qarange(-0.25, 0.25, 0.01)
    fq_list = qarange(4900, 5000, 5)
    qs_test(1, fq_list)
amp_coefficient_check(amp_coefficient=1.0)¶

更新比特能谱实验线路幅值。

驱动功率最小只能提供 -40dB，因为不同样本的线路配置不同，这个功率有时还是很高的。 所以它需要在驱动功率之外额外乘以一个幅值系数，用来进一步降低幅值。

参数

amp_coefficient (float) – 幅值系数 0-1

引发

ExperimentError – 幅值系数在 0-1 范围之外

返回类型

None

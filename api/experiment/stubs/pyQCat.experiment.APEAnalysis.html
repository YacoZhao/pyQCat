analyze()¶

APE 实验结果分析方法

使用 consine_fit 方法进行拟合，如果注册类型为 amp-phase ，则利用 amp 值进行拟合， 如果注册类型为 P0-P1 ，则利用 P0 值进行拟合

拟合公式为：

\[y = amp * cos\left(2 * π * freq * \left( x - phi \right)\right) + offset\]

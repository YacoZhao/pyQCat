find_peak(value_list, num_peak, distance)¶

查找一维数组峰值并返回峰值索引

参数

  • value_list – 需要找到峰值的数组，ndarray

  • num_peak – 数据集中的峰/谷的数量。

  • distance – 相邻峰之间所需的最小水平距离，较小的峰首先被移除，直到所有剩余的峰满足条件

返回类型

Tuple

返回

元祖类型，第一个值为int类型的峰值索引，第二个值为float类型的峰值

案例

x, y0, y1, title0, title1, file_name = self.single_qubit_usualness()
# 假设x表示扫描参数，y0表示amp实验结果，y1表示phase实验结果
index1, _ = find_peak(y0, 1, int(len(x) / 4))
index2, _ = find_peak(y1, 1, int(len(x) / 4))

参数

  • x – x数据

  • y – y数据

  • percentile – 噪声阈值

  • window_len – 平滑函数窗口大小

smooth(x, window_len=11, window='hanning')¶

对数据进行平滑处理。

这种方法是基于一个按比例的窗口进行数据平滑的方法，通过窗口大小来划分数据。

参数

  • x – 输入数据

  • window_len – 平滑化窗口的大小，保证是一个奇数

  • window – 窗口的类型，可以选择 flat , hanning , hamming , bartlett , blackman , 此处默认为 hanning

返回类型

List

返回

平滑后的信号

案例

import numpy as np

t = np.linspace(-2, 2, 40)
x = np.sin(t) + np.random.randint(len(t)) * 0.2
y = smooth(x)

plt.plot(x, label='pre smooth')
plt.plot(y, label='after smooth')
plt.legend()
plt.show()

points_to_time(points, sample_rate=1.6)¶

根据采样点的序号计算采样点的时刻

采样定理:

\[r = {\frac{p}{t}}\]

上式中 r 表示采样率，t 表示采样时间， p 表示在 t 时间内的采样点数

参数

  • points (int) – 采样点的序号

  • sample_rate (float) – 采样率，默认为1.6

返回类型

int

返回

当前采样点的时间

list_to_dict(node_list)¶

参数

node_list (list) – 列表元素

返回类型

Dict

返回

转化后的字典

案例

>>> dicts = list_to_dict(arrs)
>>> print(dicts)

pyQCat.tool.iqprobability.get_qubits_probability(discriminator_list: List[pyQCat.tool.iqprobability.IQdiscriminator], i_list: List, q_list: List, repeat: int) → List¶

联合读取获取各个状态下的概率

如果执行三比特联合读取，假设有 [q0、q1、q2]，则三种比特一共可能出现8种情况，分为为 [000, 001, 010, 011, 100, 101, 110, 111] , 获取联合读取实验结果后，用每个比特 的 IQ 分类器预测每个比特所处的状态 0 或 1, 然后统计读取结果在8种可能情况下的个数， 除以实验重复次数即的各个比特每个情况下的出现概率

参数

  • discriminator_list – 联合读取时，每个比特对应的 IQ 分类器列表

  • i_list – 联合读取 I 数据

  • q_list – 联合读取 Q 数据

  • repeat – 实验重复次数

返回

联合读取在每种状态下的概率

返回类型

List

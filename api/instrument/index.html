pyQCat.instrument¶

pyQCat.instrument 模块实现了一体机和商用仪器的通信及操作流程，用户可以直接通过调用接口实现数据传递。

基类：object

Usage:

>>>inst = INSTRUMENT()
>>>N5173B = inst['N5173B']

class pyQCat.instrument.Instrument(support_qubit=8)¶

基类：object

一体机设备封装类

实现为一体机各个模块绑定参数的功能

参数

  • support_qubit (int) – 一体机型号，目前支持8位和30位，默认为8位

  • _kernel (Kernel) – 测量参数封装类，设备对象与数据库映射之间的关联类

  • _sweep (List) – 实验最终施加到设备上的扫描参数

  • instrument_aio (InstrumentAIODoc) – 设备数据库映射类

  • is_connect (bool) – 设备是否连接的标志位

select(**kwargs)¶

选择一体机的指定模块，并打开对应的通道

一体机型号为 8bit 时：

  • Z_dc_control 通道范围为 1-8

  • Z_flux_control 通道范围为 1-8

  • XY_control 通道范围为 5-20

  • Readout_control 通道范围为 1-4

一体机型号为 30bit 时：

  • Z_dc_control 通道范围为 1-80

  • Z_flux_control 通道范围为 1-80

  • XY_control 通道范围为 9-40

  • Readout_control 通道范围为 1-8

参数

**kwargs – 字典类型，键为指定模块名，值为通道值 模块名可以为 Z_dc_control 、 Z_flux_control 、 XY_control 、 Readout_control

案例

self.inst.select(
    Z_dc_control=z_dc_channels,
    Z_flux_control=z_flux_channels,
    XY_control=xy_channels,
    Readout_control=readout_channels
)

set_sync_period(value)¶

设置一体机执行周期，在进行重复实验时，每个周期内执行一次施加波形，执行实验的操作，非反馈模式下可以用这个方法 保证一体机各个模块周期相同

参数

value –

一体机施加波形周期

  • 最小值为10

  • 最大值为1000

  • 步长为5，设置的周期值必须可能被5整除

  • 单位: 微秒(us)

sweep_delay(module: str, channel: int, start=None, stop=None, step=None, points=None, repeat=1000, index=0)¶

一体机 AWG/RF/ADDA 模块设置扫描延时

一体机型号为 8bit 时：

  • Z_flux_control 通道范围为 1-8

  • XY_control 通道范围为 5-20

  • Readout_control 通道范围为 1-4

一体机型号为 30bit 时：

  • Z_flux_control 通道范围为 1-80

  • XY_control 通道范围为 9-40

  • Readout_control 通道范围为 1-8

参数

  • module (str) – 模块名，可以设置为 Z_flux_control 、 XY_control 、 Read_out_control

  • channel (int) – 通道

  • start – 触发延时扫描参数的开始时间，范围为 0-100000ns

  • stop – 触发延时扫描参数的结束时间，范围为 0-100000ns

  • step – 触发延时扫描参数的步长，范围为 0-100000ns

  • points – 可迭代的对象，如果没有给出起始、结束、步长三个参数，可以直接传入延时列表交给 points 参数

  • repeat – 重复次数，默认为1000，只有在非反馈模式的时候有效

  • index – 延时列表的索引

引发

ArgumentsError – 如果传入的延时参数有误

注意

步长必须为 5ns 的整数倍

sweep_wave(module: str, channel, wavedata, points=None, repeat=1000)¶

一体机 AWG/RF/ADDA 社会中扫描自定义波形

一体机型号为 8bit 时：

  • Z_flux_control 通道范围为 1-8

  • XY_control 通道范围为 5-20

  • Readout_control 通道范围为 1-4

一体机型号为 30bit 时：

  • Z_flux_control 通道范围为 1-80

  • XY_control 通道范围为 9-40

  • Readout_control 通道范围为 1-8

参数

  • module (str) – 模块名，可以设置为 Z_flux_control 、 XY_control 、 Read_out_control

  • channel (int) – 通道

  • wavedata – 列表对象，迎来记录波形数据

  • points – 列表对象，记录扫描波形扫描列表

  • repeat – 重复次数，默认为1000，只有在非反馈模式的时候有效

引发

  • InvalidValueError – 一体机位数既不是8比特也不是30比特

  • TypeError – 传入的 points 参数不为列表

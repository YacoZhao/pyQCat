pyQCat.database.Kernel¶

class pyQCat.database.Kernel(qubits: int)¶

基类：object

测量参数核心操作类

Kernel 类主要负责对 MeasureAIODoc 和 SweepDoc 进行参数设置和修改功能，按照不同的实验需求， 施加不同的实验状态

参数

  • instrument (MeasureAIODoc) – 一体机测量参数数据库映射类

  • measurement (SweepDoc) – 扫描参数

  • qubits – 一体机型号，8位或30位

set_hardware_module(module: str, channel: int, name: str, value)¶

设置一体机指定模块封装参数的属性

参数

  • module –

    模块名称，你可以选择：

    • Z_flux_control

    • XY_control

    • Trig_out_control

    • Read_out_control

  • channel –

    通道值

    一体机型号为 8bit 时：

    • Z_flux_control 通道范围为 1-8

    • XY_control 通道范围为 5-20

    • Readout_control 通道范围为 1-4

    一体机型号为 30bit 时：

    • Z_flux_control 通道范围为 1-80

    • XY_control 通道范围为 9-40

    • Readout_control 通道范围为 1-8

    • Trig_out_control 通道范围为 1-4

  • name – 属性名称

  • value – 属性值

实际案例

>>> self._kernel.set_hardware_module('Trig_out_control',
>>>                                  chanenl, 'pulse_width', value)
>>> self._kernel.set_hardware_module('Read_out_control',
>>>                                  channel, 'sampling_delay', list(values))
set_waveform_module(module, channel, wave_type, wave_param)¶

设置波形

为线路模块连接上 WaveFormDoc, 即修改 XY_control 、 Z_flux_control 、 Read_out_control 、 Trig_out_control 的 waveform 属性

参数

  • module –

    模块名，其与波形对应关系如下

    • ’XY_control’: XYlineDoc,

    • ’Z_flux_control’: ZLineDoc,

    • ’Read_out_control’: ReadoutDoc,

    • ’Trig_out_control’: TrigOutDoc

  • channel – 通道值

  • wave_type –

    波形类型，可选类型为

    • 0: 表示 SineWaveDoc

    • 1: 表示 DcWaveDoc

    • 2: 表示 CustomWaveDoc

  • wave_param – 波形参数

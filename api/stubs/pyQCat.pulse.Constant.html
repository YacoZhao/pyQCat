pyQCat.pulse.Constant¶

class pyQCat.pulse.Constant(time: float, amp: float, name: str = 'Z')¶

基类：pyQCat.pulse.base_pulse.PulseComponent

Constant 波形，作用于 Z 线

一个简单的持续波形, 只有幅值和时长两个参数

\[f\left(x\right) = amp , 0 <= x < time f\left(x\right) , elsewhere\]
参数

  • amp – 幅值

  • time – 时宽

  • name – 默认为 Z

judge_background_noise(inst, qubit, compensate, dc, dcm, amp_list, drive_power, sample, judge_step=5, log_path=None)¶

Judge local noise.

参数

  • inst – Instrument object

  • qubit – Qubit object

  • compensate – PulseCorrection object

  • dc – dict,

  • dcm – discriminator

  • amp_list – list of amp, scan amp value

  • drive_power – drive power

  • sample – qubit.sample prefix

  • judge_step – change drive power step

  • log_path – save qubit info path

返回

True or False, judge local noise, True means existence local noise.

返回类型

background_noise_flag

Update qubit.XYwave.Xpi, qubit.XYwave.Xpi2, qubit.drive_drive_power. Judge Ax1 value range [0.7, 0.9], SinDamp fit. Directly, use Rabi experiment’s result of consine fit.

参数

  • inst – Instrument object

  • qubit – Qubit object

  • compensate – PulseCorrection object

  • dc – dict,

  • dcm – discriminator

  • amp_list – list of amp, scan amp value

  • sample – qubit.sample prefix

  • expect_Xpi – expectation value of Xpi

  • log_path – save qubit info path

返回

True or False, True means update successfully.

xpi_flag

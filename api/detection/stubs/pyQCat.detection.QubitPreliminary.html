select_dc_source(dc_source='qaio', qaio_ip='localhost', qaio_type=8)¶

选择 DC 源

  • 如果设置为 qaio，则选择一体机的仪器模式作为 DC 源

  • 如果设置设置为其他，则商用仪器作为 DC 源

参数

  • dc_source (str) – DC源

  • qaio_ip (str) – ip地址

  • qaio_type (int) – 一体机类型，支持8位和30位

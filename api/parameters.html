pyQCat.parameters¶

pyQCat.parameters 模块用于获取实验参数

pyQCat.parameters.get_parameters(name: str, bit: Optional[Union[int, List]] = None, root: str = 'D:\\work_loc\\pyqcat\\pyqcat_doc_ch/config', union=None)¶

获取实验所需参数

参数

  • name –

    对象名称

    • ac_crosstalk: 获取AC串扰系数

    • dc_crosstalk: 获取DC串扰系数

    • union_readout: 获取联合读取参数

    • fidelity_matrix: 获取保真度矩阵,

    • instrument: 获取设备

    • compensate: 获取波形校准数据

    • qubit: 获取Qubit对象

    • dcm: 获取IQ分类器

    • fq_zamp: 获取AC调制谱参数

    • fq_zamp_dc: 获取DC调制谱参数

    • readout2q: 获取两比特读取参数

    • cz_dict: 获取CZ门

  • bit – 比特编号

  • root – 获取参数的路径，默认为空，从数据库中加载，否则从你提供的配置文件路劲中获取

  • union –

Examples:

inst = get_parameters("instrument")
qubit = get_parameters("qubit", bit)
compensate = get_parameters("compensate", bit)
dc = corrected_dc(6)
dcm = get_parameters("dcm", bit)
pyQCat.parameters.corrected_dc(dimension: Union[int, List] = 6, root: str = 'D:\\work_loc\\pyqcat\\pyqcat_doc_ch/config', sweet_qubit: Optional[int] = None, static_qubits: Optional[List[pyQCat.qubit.qubit.Qubit]] = None) → Dict¶

获取实验中施加到各个比特上的DC电压值

如果传入 sweet_point 参数，则 sweet_point 对应的比特取 dc_max 属性， 其他比特取 dc_min 属性，且不需要进行畸变修正；

如果没有传入 sweet_point , 则所有比特电压值取比特的 dc 属性，且需要进行畸变修正

如果传入 static_qubits 参数，则从 static_qubits 中的 qubit 实例中获取电压值， 其余比特从数据库中获取

参数

  • dimension – 比特个数，默认为6比特

  • root – 根目录，用于从json中获取数据

  • sweet_qubit – 工作比特

  • static_qubits – 传参输入自选的比特

引发

TypeError – dimension 属性既不是 int 类型，也不是 List 类型

返回

每个通道对应的电压值，最终施加到比特上的电压

返回类型

dict

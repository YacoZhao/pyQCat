Experiment Parameters (pyQCat.parameters)¶

该模块包装了用于获取实验参数的通用方法

get_parameters(name, bit=None, root='D:\\\\work_loc\\\\pyqcat\\\\pyqcat_docs/config', union=None)¶

获取实验所需参数

参数

  • name (str) –

    对象名称，你可以选择：

    • ac_crosstalk: 获取AC串扰系数

    • dc_crosstalk: 获取DC串扰系数

    • union_readout: 获取联合读取参数

    • fidelity_matrix: 获取保真度矩阵,

    • instrument: 获取设备

    • compensate: 获取波形校准数据

    • qubit: 获取Qubit对象

    • dcm: 获取IQ分类器

    • fq_zamp: 获取AC调制谱参数

    • fq_zamp_dc: 获取DC调制谱参数

    • readout2q: 获取两比特读取参数

    • cz_dict: 获取CZ门

  • bit (Union[int, List, None]) – 比特编号

  • root (str) – 获取参数的路径，默认为空，从数据库中加载，否则从你提供的配置文件路劲中获取

  • union – 获取联合IQ分类器的时候使用，默认为空

案例

inst = get_parameters("instrument")
qubit = get_parameters("qubit", bit)
compensate = get_parameters("compensate", bit)
dc = corrected_dc(6)
dcm = get_parameters("dcm", bit)

# 获取联合读取分类器
dcm_list = [get_parameters("dcm", i, union=union_bit_list) for i in union_bit_list]
corrected_dc(dimension=6, root='D:\\\\work_loc\\\\pyqcat\\\\pyqcat_docs/config', sweet_qubit=None, static_qubits=None)¶

获取实验中施加到各个比特上的DC电压值

如果传入 sweet_point 参数，则 sweet_point 对应的比特取 dc_max 属性， 其他比特取 dc_min 属性，且不需要进行畸变修正；

如果没有传入 sweet_point , 则所有比特电压值取比特的 dc 属性，且需要进行畸变修正

如果传入 static_qubits 参数，则从 static_qubits 中的 qubit 实例中获取电压值， 其余比特从数据库中获取

参数

  • dimension (Union[int, List]) – 比特个数，默认为6比特

  • root (str) – 根目录，用于从json中获取数据

  • sweet_qubit (Optional[int]) – 工作比特

  • static_qubits (Optional[List[Qubit]]) – 传参输入自选的比特

引发

TypeError – dimension 属性既不是 int 类型，也不是 List 类型

返回类型

Dict

返回

每个通道对应的电压值，最终施加到比特上的电压

update_qubit_dc(bit, vol, root='D:\\\\work_loc\\\\pyqcat\\\\pyqcat_docs/config')¶

更新配置文件中 qubit.json 中的比特电压值

参数

  • bit (int) – 比特编号

  • vol (float) – 电压值

  • root (str) – 配置文件 qubit.json 所在位置的绝对路径

返回类型

None

decorate_dc(q_list, shape=(0, 5))¶

利用DC串扰系数修正电压

参数

  • q_list (List[Qubit]) – 比特列表

  • shape (Tuple) – 比特规模

返回类型

Dict

返回

经过串扰修正后施加到各个比特上的电压

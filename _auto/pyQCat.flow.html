pyQCat.flow.common module¶

make_graph(schema)¶

Construct the task graph (dag) from a given schema.

Parses the graph schema definition and creates the task graph. Tasks are the vertices of the graph and the connections defined in the schema become the edges.

A key in the schema dict represents a parent task and the value one or more children:

{parent: [child]} or {parent: [child1, child2]}

The data output of one task can be routed to a labelled input slot of successor tasks using a dictionary instead of a list for the children:

{parent: {child1: ‘positive’, child2: ‘negative’}}

An empty slot name or None skips the creation of a labelled slot:

{parent: {child1: ‘’, child2: None}}

The underlying graph library creates nodes automatically, when an edge between non-existing nodes is created.

参数

schema (dict) – A dictionary with the schema definition.

返回

A reference to the fully constructed graph object.

返回类型

DiGraph

引发

DirectedAcyclicGraphUndefined – If the schema is not defined.

Automated Testing Flow (pyQCat.flow)¶

pyQCat.flow 模块是对 pyQCat.experiment 模块的扩展，将各个实验组合形成量子比特自动化测试及校准的流程，形成节点输出， 用户可以通过自由组合 node 节点，实现自定义的自动化测试流程。

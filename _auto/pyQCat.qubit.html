class Qubit(*args)¶

基类：object

量子比特实体类

量子比特的基类，包含自动化检测及校准流程的所有参数

参数

  • bit (int) – 比特编号，用来区别不同的比特

  • tunable (bool) – 比特是否可以调制标志位

  • goodness (bool) – 比特是否可以正常工作标志位

  • probe_freq (float) – 比特的读取频率

  • probe_power (float) – 比特的读取功率

  • drive_freq (float) – 比特的驱动频率

  • drive_power (float) – 比特的驱动功率

  • tls_freq (float) – TODO

  • anharmonicity (float) – TODO

  • dc (float) – 将比特调制到最佳工作点的 DC 电压

  • dc_max (float) – 施加到比特上的最大电压

  • dc_min (float) – 施加到比特上的最小电压

  • XYwave (XYwave) – XY 方向上的波形，一体机 RF 模块施加

  • Zwave (Zwave) – Z 方向上的波形，一体机 AWG 模块施加

  • Mwave (Mwave) – 测量波形，一体机 ADDA 模块施加

  • T1 (float) – 量子比特纵向弛豫时间

  • T2 (float) – 量子比特横向弛豫时间

  • sample (str) – 比特标签，用于生成结果路径

  • xy_channel (int) – XY 方向上波形施加的通道，对应一体机 RF 模块启动通道

  • z_flux_channel (int) – Z 方向上波形施加的通道，对应一体机 AWG 模块启动通道

  • z_dc_channel (int) – DC 电压施加通道，对应一体机 DC 模块启动通道

  • readout_channel (int) – 读取通道，对应一体机 ADDA 模块启动通道

  • sample_delay (int) – 当前比特读取通道的采样延时

  • sample_width (int) – 当前比特读取通道的采样带宽

  • union_readout (union_readout) – 当联合读取时，传入特定的波形

import_qubit(json_file=None)¶

导入比特参数，更新 Qubit 对象属性

当提供了 json_file 路径时，会加载指定 json 文件，将文件中的属性加载到当前的 Qubit 对象上

如果没有提供 json_file ，则会去数据库中查询最近的 QubitStoreDoc 记录， 如果查到结果了，将 QubitStoreDoc 对象参数更新到当前的 Qubit 对象上， 如果没有查到结果，则会创建一个全新的 Qubit 对象

参数

json_file (Optional[str]) – 指定 josn 文件位置

返回类型

None

get_qubit_by_time(create_time, bit=None)¶

从数据库中加载最新的 QubitStoreDoc 对象，将参数更新到当前的 Qubit 属性中

参数

  • create_time – 创建时间，如 2021-03-26 15:23:38.442

  • bit (Optional[int]) – 比特编号

返回类型

None

export_qubit(export_path=None)¶

导出 Qubit 参数信息，生成 yaml 文件保存

参数

export_path (Optional[str]) – 导入比特信息后的 yaml 文件存储路径，默认为 `config/qubit_data’

返回类型

None

log_qubit(msg, log_path=None)¶

记录比特日志信息，可以通过此方法记录比特运行状态

参数

  • msg (str) – 记录的日志信息

  • log_path (Optional[str]) – 日志保存路径

返回类型

None

save_qubit()¶

将比特参数信息保存到数据库中，如果比特参数发生了改变，则会在数据库中创建新的 QubitStoreDoc 对象

save_by_yaml(yaml_path)¶

解析 yaml 文件中记录 Qubit 参数信息，然后加载到数据库中

参数

yaml_path (str) – yaml 文件路径，例如 {path}/config/q0.yaml

返回类型

None

reset()¶

比特初始化操作，如果你意外更新 Qubit 的某些参数，你可以用此方法，让比特恢复成初始状态

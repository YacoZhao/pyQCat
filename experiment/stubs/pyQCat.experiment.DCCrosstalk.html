基类：object

DC 串扰实验

DC Crosstalk理论上可以采用和AC Crosstalk类似的方法，主要问题在于DC Crosstalk 会引起比特静态工作点偏移，造成pulse无法共振驱动比特，以及会造成读取判据发生变化。 但是在一段偏差范围内，读取判据和ramsey实验还是可以正常进行。如果要规避这样的问题， 可以通过预测的目标比特电压，来动态调节比特的读取频率，让比特一直处于静态工作点。

run()¶

运行DC串扰实验

实验步骤：

  • 获取目标比特的静态工作点电压 target_qubit.dc_max ,记作 v_target

  • 更改编制比特电压，修改 bias_qubit.dc_min 属性

  • 利用实验的 correct_dc 属性，将修改的电压值作用到比特上

  • 目标比特保持当前驱动频率不变，执行Ramsey实验，计算出当前目标比特频率f10

  • 根据DC调制谱参数，计算出将比特调制到当前的f10，所需要施加的DC电压值，即为 v_real

  • 有了 v_real 、 v_target 和 init__v_bias 三个参数之后，可以预测串扰系数为K

  • 根据 freq_threshold 属性确定期望将目标比特频率调制的范围

  • 利用DC谱参数的和预计的DC串扰系数确定编制比特扫描的电压列表，记作 v_bias_list

  • 遍历 v_bias_list ,利用预测的DC串扰系数K值，计算当前 v_bias 可能对应的 v_predict

  • 在用DC调制谱公式计算当前 v_predict 下的比特频率，调整比特频率，保证比特一直处于静态工作点

  • 执行Ramsey实验，计算比特频率f10

  • 根据DC谱公式，计算比特频率真实的电压 v_real_list

  • 根据 v_bais_list 和 v_real_list 即可求出DC串扰系数

Examples:

from pyQCat.experiment.dc_crosstalk import DCCrosstalk
from pyQCat.parameters import get_parameters
from pyQCat.tool.utlis import qarange, connect_server, disconnect_server

def test_dc_crosstalk(target_bit: int, bias_bit: int,
                      init_fringe: float, delay_list: list, bit_list: list):
    inst = get_parameters("instrument")
    target_qubit = get_parameters("qubit", target_bit)
    bias_qubit = get_parameters("qubit", bias_bit)
    dc_spectrum_params = get_parameters("fq_zamp_dc", target_bit)
    compensate = get_parameters("compensate", target_bit)
    dcm = get_parameters("dcm", target_bit)
    dc_exp = DCCrosstalk(inst, target_qubit, bias_qubit, init_fringe,
                         delay_list, dc_spectrum_params, bit_list,
                         compensate=compensate,discriminator=dcm)
    dc_exp.run()

if __name__ == '__main__':
    connect_server(ip="127.0.0.1")
    delay_list = qarange(200, 500, 5)
    init_fringe = -30
    bit_list = [1, 2, 3, 4, 5, 6]
    test_dc_crosstalk(0, 1, init_fringe, delay_list, bit_list)
    disconnect_server()
run_ramsey(v_bias, fringe, fd, IF)¶

Excute ramsey experiment to get the actual voltage applied to target qubit.

参数

  • v_bias (float) – the bias qubit dc_min value.

  • fringe (float) – Ramsey oscillation frequency which is used to amplify the difference between the drive frequency and the qubit real frequency.

  • fd (float) – The drive frequency of target qubit.

  • IF (float) – The intermediate frequency of pulse.

返回

The real voltage on target qubit.

返回类型

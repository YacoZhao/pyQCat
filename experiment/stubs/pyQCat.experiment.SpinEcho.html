基类：pyQCat.experiment.top_experiment.TopExperiment

SpinEcho实验

SpinEcho实验和Ramsey实验的作用是一样的，均用来测量比特退相干时间和比特频率校准。 区别在于SpinEcho实验在两个 half pi pulse 之间加了一个 pi pulse

run(delay_list, fringe, zamp=None, schedule_index=0)¶

SpinEcho实验执行函数

参数

  • delay_list (List) – 扫描延时列表

  • fringe (float) – 初始的fringe频率

  • zamp (Optional[float]) – z线幅值

  • schedule_index (int) – 绘制波形时序图的序号

Examples:

from pyQCat.experiment.spinecho import SpinEcho
from pyQCat.parameters import get_parameters, corrected_dc
from pyQCat.tool import qarange, remote_connection

@remote_connection("127.0.0.1")
def spinecho_exp_test(bit: int, delay_list: list, fringe: float):
    inst = get_parameters("instrument")
    qubit = get_parameters("qubit", bit)
    compensate = get_parameters("compensate", bit)
    dc = corrected_dc(6, sweet_qubit=bit)
    dcm = get_parameters("dcm", bit)

    spinecho_exp = SpinEcho(inst, qubit, correct_dc=dc, compensate=compensate, discriminator=dcm)
    spinecho_exp.run(delay_list, fringe, schedule_index=20)

if __name__ == '__main__':
    delay = qarange(0, 200, 5)
    fringe = 40
    spinecho_exp_test(0, delay, fringe)

基类：pyQCat.experiment.ac_spectrum.ACSpectrum

DC 谱实验

DC调制谱理论上可以和AC调制谱一样用Ramsey实验进行测试，主要问题在于 改变比特静态工作点会造成pulse无法共振驱动比特，以及会造成读取判据发 生变化。但是在一段偏差范围内，读取判据和Ramsey实验还是可以正常进行。

Examples:

from pyQCat.experiment.dc_spectrum import DCSpectrum
from pyQCat.parameters import get_parameters, corrected_dc
from pyQCat.tool.utlis import qarange, connect_server, disconnect_server

def test_dc_spectrum(bit: int, fringe: float, delay_list: list, z_amp_list: list):
    inst = get_parameters("instrument")
    qubit = get_parameters("qubit", bit)
    compensate = get_parameters("compensate", bit)
    dc = corrected_dc(6, sweet_qubit=bit)
    dcm = get_parameters("dcm", bit)

    dc_exp = DCSpectrum(inst, qubit, fringe, delay_list,
                        correct_dc=dc, compensate=compensate, discriminator=dcm)
    dc_exp.run()

if __name__ == '__main__':
    connect_server(ip="127.0.0.1")
    delay = qarange(200, 500, 5)
    fringe = -30
    test_dc_spectrum(0, fringe, delay)
    disconnect_server()

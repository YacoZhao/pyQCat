pyQCat.experiment.T2¶

class T2(inst, qubit, correct_dc=None, compensate=None, discriminator=None, exp_name='ramsey', time_threshold=(0.3, 0.5), repeat=1000)¶

基类：object

T2实验

T2实验用来获取量子比特横向弛豫时间

T2时间描述的是qubit的在XY方向能量损耗快慢，T2实验可以借用Ramsey实验来获取， 通过扫描两个 half pi pulse 之间的延时时间，来获取Ramsey振荡信息，通过 振荡信息获取 T2 时间信息

设备对象，封装了实验相关参数的数据库映射对象，集成了各种数据交互操作

correct_dc¶

最终施加到比特上的电压，字典的键表示设备通道，字典的值为施加的电压值

compensate¶

脉冲校准对象，用于加载到比特对象上，默认为空，当设置参数时，它意味着 Z 通道打开 并且进行波形畸变、延时等修正

Type

run(delay_list, fringe, zamp=None, fq_offset=None)¶

运行T2实验

参数

  • delay_list (List) – 扫描延时列表

  • fringe (float) – 初始 fringe 频率

  • zamp (Optional[float]) – Z线幅值

  • fq_offset (Optional[float]) – 比特频率偏移值

  • 执行Ramsey或者SpinEcho实验，获取T2时间

  • 如果T2时间在扫描时长的0.3-0.5之间，则T2符合要求，返回

  • 如果T2时间小于扫描时长的0.3倍，则降低扫描时长

  • 如果T2时间大于扫描时长的0.5倍，则增加扫描时长

  • 累计5次获取不到合适的T2时间，则从前5次实验中选取最优结果作为T2返回

Examples:

from pyQCat.experiment.t2 import T2
from pyQCat.parameters import get_parameters, corrected_dc
from pyQCat.tool import qarange, remote_connection

@remote_connection("127.0.0.1")
def t2_test(bit: int, delay_list: list, fringe: float):
    inst = get_parameters("instrument")
    qubit = get_parameters("qubit", bit)
    compensate = get_parameters("compensate", bit)
    dc = corrected_dc(6, sweet_qubit=bit)
    dcm = get_parameters("dcm", bit)

    t2_exp = T2(inst, qubit, correct_dc=dc, compensate=compensate, discriminator=dcm, exp_name='ramsey')
    t2_exp.run(delay_list, fringe)

if __name__ == '__main__':
    delay = qarange(200, 20000, 200)
    fringe = 466.667 - 0.5
    t2_test(0, delay, fringe)

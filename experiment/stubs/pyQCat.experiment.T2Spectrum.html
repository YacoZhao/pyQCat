pyQCat.experiment.T2Spectrum¶

class T2Spectrum(inst, qubit, init_fringe, delay_list, fq_list, correct_dc=None, compensate=None, discriminator=None, exp_name='ramsey', repeat=1000, T2_bound=30000.0, osc_limit=0.2)¶

基类：object

T2谱实验

T2谱实验用来获取量子比特频率和T2时间之间的关系。 实验通过 zamp 将比特频率调开，然后执行Ramsey实验获取T2时间，这样就可以得到一组比特频率和T2时间的对应关系

参数

  • inst (Instrument) – 设备对象，封装了实验相关参数的数据库映射对象，集成了各种数据交互操作

  • qubit (Qubit) – 量子比特对象，指明当前实验对象

  • init_fringe (float) – 初始 finge 频率

  • delay_list (Union[List, np.ndarray]) – 扫描延时列表

  • fq_list (Union[List, np.ndarray]) – 扫描比特频率列表

  • correct_dc (Optional[Dict]) – 最终施加到比特上的电压

  • compensate (PulseCorrection) – 脉冲校准对象，用于加载到比特对象上，默认为空，当设置参数时，它意味着 Z 通道打开 并且进行波形畸变、延时等修正

  • discriminator (Optional[IQdiscriminator]) – IQ 分类器，用来将实验测量结果进行分类 从而以不同状态的概率呈现，如 P0-P1 。如果此参数为空，则会用 amp-phase 的方式展示实验测试结果

  • exp_name (str) – 获取 T2 时间的实验名称, 可以选择 ramsey 或 spinecho

  • repeat (int) – 实验重复次数

  • T2_bound (float) – T2 时间限制，为了避免错误的拟合结果导致 T2 时间过大

  • osc_limit (float) – 拟合频率限制

run()¶

执行T2谱实验

  • 根据扫描的比特频率列表计算出扫描的Z线幅值列表

  • 遍历幅值列表，将比特频率调开

  • 执行T2实验，获取T2时间，得到一组比特频率和T2的对应关系

  • 绘制图片

Examples:

import numpy as np

from pyQCat.parameters import get_parameters, corrected_dc
from pyQCat.tool import qarange, connect_server, disconnect_server
from pyQCat.experiment.t2_spectrum import T2Spectrum

def test_t2_spectrum(bit: int, fringe: float, delay_list: list):
    inst = get_parameters("instrument")
    qubit = get_parameters("qubit", bit)
    compensate = get_parameters("compensate", bit)
    dc = corrected_dc(6, sweet_qubit=bit)
    dcm = get_parameters("dcm", bit)
    fq_array = np.arange(qubit.drive_freq, qubit.drive_freq - 600, -20)
    fq_list = list(fq_array)
    t2_exp = T2Spectrum(inst, qubit, fringe, delay_list, fq_list, correct_dc=dc,
                        compensate=compensate, discriminator=dcm, exp_name='ramsey')
    t2_exp.run()

if __name__ == '__main__':
    connect_server(ip="127.0.0.1")
    delay = qarange(200, 10000, 200)
    fringe = -0.5
    test_t2_spectrum(0, fringe, delay)
    disconnect_server()

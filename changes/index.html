版本记录¶

version 0.0.2¶

日期：2021年7月20日

  1. 更新版本，新增flow流程的GUI界面执行方法

  2. 改变T2实验策略： 保证实验结果T2在扫描总时长的30%-50%之间，动态调整扫描时长，fringe频率， 找出最佳的T2，按照上述策略新增T2实验接口；

  3. 更新T2Spetrum的代码结构： 从直接调用Ramsey，改为调用T2，保证每次获取可信的T2值；

  4. 修复 Bug， pi pulse 直接将预测 drive power 写入数据库： 对预测进行判断，是否在区间 [ -40, -10]，若是则保存，否则不保存；

  5. 优化 Rabi 实验获取 Xpi: 当最小值或最大值是最后一个值，则选取最后一个值对应的 amp 为 Xpi；

  6. 更新畸变测试流程： 注释畸变测试之前 T2 值的测量代码；

  7. 优化画图功能： 优化图片的字体，大小等等；

  8. 增加 rabi_sweep_width 节点： 增加 Rabi 扫描 width 节点，目前只有接口，具体逻辑待处理；

  9. 调整比特频率的测试思路，从低频率开始扫描，判断噪声并找出f10，并根据测试需求选择是否测试f02_2

  10. 调整了比特频率中寻峰算法的计算，加入了平滑数据和寻峰条件的判断

version 0.0.3¶

日期：2021年7月26日

  1. 调整腔调制实验的拟合方式

  2. 调整腔调制实验超出最大值范围时自动调整电压扫描范围

  3. 调试 T1 谱实验： 调通 T1 谱实验，增加 P1 拟合数据画图；

  4. 调试 rabi_sweep_width 节点： 对 Rabi 实验结果 amp 进行频谱分析，频谱中如有异常值，表示振荡；

  5. 优化自动化校准GUI界面按钮点击效果

  6. 新增RabiSweepWidth参数选项设置

  7. 依赖库版本限制，scipy==1.6.0, scikit-learn=0.21.2

version 0.0.4¶

日期：2021年8月4日

  1. 解决自动化测试界面中，test_bit为list时不起作用

  2. 自动化测试flow界面中，增加全选和全不选的按钮功能

  3. 优化 XYZTiming 实验：增加 xy line 固定延时可设置参数 const_delay

  4. 自动化测试界面中， XYZTiming 实验新增const_delay参数选项

  5. 修复DC 谱实验 bug：调试 DC 谱实验，解决参数 update_step 已被替换为 expect_Xpi 问题

  6. 优化AC 串扰数据保存路径: AC 串扰实验数据，以目标比特作为父级目录，目标比特和偏置比特组合生成子目录；

  7. 自动化测试界面中，DC Crosstalk实验中新增 dc 调制谱参数

version 0.0.5¶

日期：2021年8月12日

  1. 在比特能谱实验中，使用chirp变频方案替代原不变频方案，增大比特频谱带宽，使得粗扫效率提升10倍；

  2. 增加功能：绑定比特 XYwave drive_IF 与 instrument XY_control IF 值, exp.yaml 文件 system 增加参数 drive_IF；

  3. 优化Rabi 获取Xpi：修改 Rabi 获取 Xpi 逻辑，amp-phase, 优先使用amp， 获取 amp 第一周期极大值点；

  4. flow自动化测试GUI界面system模板新增drive_IF属性；

  5. 网分接口加入outputON和outputOFF函数。

version 0.0.6¶

日期：2021年8月13日

  1. DC 谱优化：子实验目录名称优化，根据当前dc 或 dc_max 值标记生成；

  2. T1 谱优化：实验数据 Z_amp, Freq, T1 保存为 dat 文件，子实验目录名称优化，根据 zamp 值标记生成；

  3. 新增功能：在执行完flow流程之后，会在原本的实验结果图片存放路径的根路径生成一个展示结果的markdown文件

  4. 修改BUG：flow GUI执行界面 dcm_flag 属性不支持 False 输入 （已解决）

version 0.0.7¶

日期：2021年8月20日

实验结果解析展示

  1. 解决了显示数据的小数位数以及数据变化标注不准确的问题。优化了数据与实验图片的对应逻辑。

  2. 优化各flow图片收集类的结构，对重复部分进一步抽取，各目录对应关系通过类名区分作为单独参数统一独立出来。

  3. 各实验数据收集装饰器入口优化，减少冗余代码。

  4. qubit_data输出路径优化，按照bit分类

数据处理方案优化

  1. chirp变频范围由 [IF， IF+bandwidth] 调整为 [IF-bandwidth/2， IF+bandwidth/2]，拟合结果不需要做bandwidth/2的修正。

  2. Rabi振荡检测中的MAD范围扩大因子b由1.482调整为2.5。

实验逻辑优化

  1. 调整寻找比特频率的逻辑，结合chrip方案、寻峰算法及raibScanWidth，大幅度提升比特频率的查找准确率

  2. yaml文件增加sweet point选项供用户选择，自由选择DC串扰校准的制备点

  3. goodness节点加入拟合Q值功能

自动化测试GUI界面更新

  1. flow GUI界面添加sweet point参数选项
